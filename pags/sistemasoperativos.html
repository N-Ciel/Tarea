Sistemas Operativos

Un sistema operativo (SO) es el software principal o conjunto de programas de un sistema informático que gestiona los recursos de hardware y provee servicios a los programas de aplicación.

En términos sencillos, es el intermediario entre tú (el usuario), las aplicaciones que usas (como VS Code o un navegador) y las piezas físicas de la computadora (procesador, memoria, disco duro). Sin un sistema operativo, tendrías que escribir código complejo cada vez que quisieras guardar un archivo o imprimir una hoja.

Se encarga de decidir qué programa usa el procesador y durante cuánto tiempo. Esto permite que sientas que la computadora hace varias cosas a la vez (multitarea). Asigna espacio de memoria a cada aplicación para que no se "pisen" entre ellas. Cuando cierras un programa, el SO libera ese espacio. Controla cómo se guardan, leen y borran los datos en el disco duro o SSD a través del sistema de archivos. Gestiona la comunicación con periféricos como el teclado, el mouse, la impresora o el monitor mediante controladores (drivers). Proporciona la forma en que interactuamos con la máquina, ya sea visual (iconos y ventanas) o mediante texto (consola de comandos).

Por que es importante?

Como estudiante de informática, entender los sistemas operativos es clave porque te enseña cómo se gestionan realmente los recursos. Conceptos como la automatización con Bash en Linux, el manejo de permisos de archivos y la gestión de procesos son el "pan de cada día" en el desarrollo de software y la administración de sistemas.
